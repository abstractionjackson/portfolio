# Portfolio
The software engineering portfolio of San Francisco Developer Jackson Galan, [portfolio](https://github.com/abstractionjackson/portfolio) serves as a means to contact the developer, peruse his body of work, and learn more about his skills and experience.

## Hugo
The site itself is built on the Hugo framework, a fast way to build static web sites, and manage their content.

## GitHub Pages
The site is deployed to GitHub Pages, a zero-cost means to host static websites that integrates coveniently with Hugo via GitHub Actions.
